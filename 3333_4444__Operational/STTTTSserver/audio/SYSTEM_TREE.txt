QUICK START
===========

1. Install:
   cd /path/to/audio
   npm install

2. Start:
   ./start-audio-server.sh

3. Access:
   http://localhost:3030/audio-player-visualization.html

4. Test:
   ffmpeg -re -f lavfi -i "sine=frequency=440:duration=60" \
     -ar 48000 -ac 1 -acodec pcm_s16le -f rtp rtp://localhost:3333

API ENDPOINTS
=============

GET  /health ................. Server health check
GET  /api/streams ............ List all streams
GET  /api/streams/:port ...... Stream details
POST /api/streams/:port/start  Start stream
POST /api/streams/:port/stop . Stop stream
